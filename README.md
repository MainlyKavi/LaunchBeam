# LaunchBeam landing page

Premium light-mode marketing website for LaunchBeam, a waitlist and pre-launch
platform for startup founders.

## What is included

- Minimal Apple-inspired light liquid-glass design system
- Interactive waitlist page builder simulation
- Template showcase with switchable product categories
- Referral, analytics, validation score, customization, workflow, pricing, and
  final CTA sections
- Responsive mobile navigation and layouts
- Accessible validation, save, copy, focus, and reduced-motion states
- Host-aware Open Graph and X metadata with a bespoke social preview image
- Smoke tests for server-rendered content and starter cleanup

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run typecheck
npm test
```

This draft intentionally does not include authentication, payments, database
storage, or real email collection.
